# next-detect-os
> Detect os for next.

## usgage:
```js
import NxDetectOs from 'next-detect-os';

// is:
NxDetectOs.isIos();
NxDetectOs.isAndroid();
NxDetectOs.isBlackBerry();
NxDetectOs.isMac();
NxDetectOs.isWindows();
NxDetectOs.isLinux();
NxDetectOs.isWechat();
NxDetectOs.isMobile();

// get os: 
NxDetectOs.get();
// [ios/android/blackBerry/mac/windows/linux/wechat]
```
